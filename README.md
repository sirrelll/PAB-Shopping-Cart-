# PAB-Shopping Cart


Tugas membuat aplikasi belanja sederhana pakai Flutter. Aplikasinya ada 3 halaman yaitu halaman produk, keranjang, dan checkout.

---

## Yang Sudah Dibuat
- Tambah produk ke keranjang
- Lihat isi keranjang sama quantity-nya
- Bisa nambah atau ngurangin quantity
- Bisa hapus item dari keranjang
- Total harga tampil otomatis
- Search produk by nama
- Filter produk by kategori
- Halaman checkout ada ringkasan pesanan + form isi data

## Isi File

Semua kode ada di `lib/main.dart`. Dibagi jadi beberapa bagian:

- **HalamanProduk** - nampilin list produk, ada search dan filter kategori
- **HalamanCart** - isi keranjang, bisa update qty dan hapus
- **HalamanCheckout** - ringkasan belanja + form nama, alamat, nomor hp
- **fungsi rupiah()** - buat ngubah angka jadi format Rp
